## Краткое описание
Одностраничный, адаптивный сайт, который создан с использованием HTML и CSS.   
Реализован по правилам flexbox, используется CSS анимация, медиазапросы, подключены шрифты.
#### Стек технологий
  * HTML 5;
  * CSS 3;
#### С применением
  * Flexbox;
  * Media queries
  * Семантическая разметка;
